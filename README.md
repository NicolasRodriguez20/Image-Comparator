# Image-Comparator
Aplicación de escritorio para detectar imágenes duplicadas mediante hashes perceptuales e interfaz gráfica en Tkinter.

## Características

- Detección automática de imágenes duplicadas
- Vista previa de miniaturas
- Eliminación segura mediante papelera de reciclaje
- Interfaz gráfica moderna desarrollada con Tkinter

## Tecnologías

- Python
- Tkinter
- Pillow
- ImageHash
- Send2Trash

## Estado

Proyecto funcional y publicado.
